## Project Overview

This is a modern React application built with:
- **Vite** as the build tool (fast, modern alternative to Create React App)
- **React 19** with the new React Compiler
- **TanStack Router** for type-safe file-based routing
- **TanStack Query** for data fetching and caching
- **Tailwind CSS 4** for styling
- **TypeScript** for type safety

## Running the Application

### Development
```bash
cd client
npm run dev
```
Opens browser automatically at `http://localhost:3000`

### Production Build
```bash
npm run build
npm run preview
```

### With Backend
The Spring Boot backend runs on port 8080. Both can run simultaneously:

Terminal 1 (Backend):
```bash
cd server
./mvnw spring-boot:run
```

Terminal 2 (Frontend):
```bash
cd client
npm run dev
```
